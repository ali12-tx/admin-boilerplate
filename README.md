# Admin Panel

This project is a Vite + React + TypeScript admin panel scaffolded with shadcn-ui and Tailwind CSS.

## Getting started

- Ensure Node.js and npm are installed (recommended via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).
- Install dependencies: `npm install`
- Start the development server: `npm run dev`
- Run a production build: `npm run build`
- Preview the production build locally: `npm run preview`

## Project structure and tech

- Build tooling: Vite with React SWC
- Styling: Tailwind CSS and shadcn-ui components
- Forms and validation: React Hook Form, Zod, @hookform/resolvers
- Data fetching: @tanstack/react-query
- Routing: react-router-dom
